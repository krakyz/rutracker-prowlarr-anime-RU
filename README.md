# rutracker-prowlarr-anime-RU

Описание торрент-трекера [rutracker.org](https://rutracker.org/) в формате [Prowlarr Cardigann YML](https://wiki.servarr.com/prowlarr/cardigann-yml-definition), предназначенное исключительно для аниме-разделов. 

Приводит заголовки раздач в вид, [рекомендованный TRaSH-Guides](https://trash-guides.info/Sonarr/Sonarr-recommended-naming-scheme/#episode-format). Выносит все не-romanji названия из заголовка в конец.

Некоторые релизы переводятся TheTVDB довольно вольно, и возможности их найти внутри Sonarr нет. Пример: "Love, Chunibyo & Other Delusions!". На Rutracker только одна раздача имеет это название среди альтернативных, и только по этому названию она совпадает с алиасами TheTVDB. Так на некоторых тайтлах вы получите неполный поиск (вероятнее всего на тех, у которых необычное или вычурное название). На данный момент поменять TheTVDB на что-то более актуальное или практичное нельзя. Чтобы в поиск попадало хоть что-то, все альтернативные названия вынесены в скобки в конце заголовка.

Сплит-куры скорее всего корректно не найдёт, хоть я и пытался добавить обрабокту через "Part / Часть".

В Series Type = Anime лично у меня поиск бесконечный, поэтому работал исключительно в режиме Standart.
